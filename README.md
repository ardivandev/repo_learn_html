## 📖 Penjelasan Materi

### 1. Heading

Belajar menggunakan tag heading seperti:

```html
<h1> sampai <h6>
```

Digunakan untuk membuat judul dan subjudul pada halaman website.

---

### 2. Paragraph

Belajar membuat paragraf menggunakan:

```html
<p>
```

---

### 3. Text Formatting

Mempelajari format teks seperti:

- Bold
- Italic
- Underline
- Mark
- Small
- Delete

Contoh:

```html
<b>Bold</b>
<i>Italic</i>
```

---

### 4. Quotations

Belajar membuat kutipan menggunakan:

```html
<blockquote>
<q>
```

---

### 5. Link

Membuat hyperlink menggunakan:

```html
<a href="">
```

---

### 6. Gambar

Menampilkan gambar menggunakan:

```html
<img src="" alt="">
```

---

### 7. Tabel

Belajar membuat tabel HTML menggunakan:

```html
<table>
<tr>
<td>
<th>
```

---

### 8. List

Mempelajari daftar pada HTML:

- Ordered List (`<ol>`)
- Unordered List (`<ul>`)
- Description List (`<dl>`)

---

### 9. Div & Span

Belajar pengelompokan elemen menggunakan:

```html
<div>
<span>
```

---

### 10. Iframe

Menampilkan halaman atau video dari website lain menggunakan:

```html
<iframe>
```

---

### 11. Semantik HTML

Belajar struktur semantic HTML seperti:

```html
<header>
<nav>
<main>
<section>
<article>
<footer>
```

Agar struktur website lebih rapi dan mudah dipahami browser.

---

### 12. Form

Belajar membuat form input pengguna seperti:

```html
<form>
<input>
<textarea>
<select>
<button>
```

---

### 13. Media (Video & Audio)

Belajar menampilkan media pada website menggunakan:

```html
<video>
<audio>
```

Contoh:

```html
<video controls>
    <source src="video.mp4" type="video/mp4">
</video>

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>
```
