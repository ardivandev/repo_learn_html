List

Digunakan untuk pembuatan list, dengan 3 jenis list yaitu : unordered list, ordered list dan description list

Contohnya :

Unordered List adalah list yang tidak memakai urutan angka

  • HTML
  • CSS
  • JavaScript

Ordered List adalah list yang memakai urutan angka

  1. Bangun tidur
  2. Sarapan
  3. Olahraga
  4. Belajar

Description List adalah list yang digunakan untuk membuat daftar istilah dan penjelasannya.

HTML
Hyper Text Markup Language adalah bahasa yang digunakan dalam pembuatan struktur pada sebuah website
CSS
Cescading Style Sheet adalah sebuah bahasa yang digunakan untuk menghias atau mendekorasi sebuah website
